En este proyecto en Power BI se utilizó una base de datos de una empresa de retail internacional. Los datos de las diversas tablas como productos, sucursales, ventas y clientes fueron obtenidos de 
diversas fuentes como hojas de texto, libros de excel, formatos csv y páginas web. Se utilizaron medidas DAX para manipular los datos existentes y obtener insights valisos como clientes con mayor 
numero de compras, productos que representan más porcentaje de las ventas totales, promedios, máximos y mínimos de compras por día. 
Se crearon dashboards interactivos utilizando gráficos con diferentes opciones de visualización, filtros, segmentadores, botónes y KPI para la mejor comprensión de los datos y sus resultados.
